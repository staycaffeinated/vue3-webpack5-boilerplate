# Vue 3,  Webpack 5 Boilerplate

A project boilerplate for Vue3, Webpack 5.


## Utilizes 

- Vue 3.2
- Vue Router 4.0
- Vuex 4.0
- Typescript 3.9
- Babel 7.16
- Webpack 5.64


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
